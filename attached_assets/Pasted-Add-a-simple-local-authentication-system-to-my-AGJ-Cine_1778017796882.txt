1. CREATE `src/contexts/AuthContext.tsx`:
   - Create a React context with these values:
     - `user`: object `{ username: string }` or `null`
     - `isLoggedIn`: boolean
     - `login`: function that takes (username: string, password: string) and returns boolean
     - `logout`: function
     - `register`: function that takes (username: string, password: string) and returns boolean
   - On mount, check `localStorage.getItem('agj_user')` to restore session.
   - `login` function:
     - Get users from `localStorage.getItem('agj_users')` (parse JSON, default to empty array).
     - Find user where username AND password match.
     - If found, save to `localStorage.setItem('agj_user', JSON.stringify(user))` and return true.
     - If not found, return false.
   - `register` function:
     - Get users from localStorage.
     - Check if username already exists. If yes, return false.
     - Add new user `{ username, password }` to the array.
     - Save back to localStorage.
     - Auto-login the new user (save to agj_user).
     - Return true.
   - `logout` function:
     - Remove `agj_user` from localStorage.
     - Set user state to null.
   - Export `useAuth` hook that returns the context.
